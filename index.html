About CantoRocks

CantoRocks is an interactive HTML5 game designed to help you practice spelling Cantonese words using Jyutping romanization and reinforce your basic Cantonese vocabulary.

Game Modes

  • Normal Mode: Type Cantonese words without tone marks - letters only
  • Full Mode: Type complete Jyutping including tone numbers

How to Play

  1. Choose your game mode above
  2. Press Enter to start the game
  3. Type the Cantonese words shown on the falling rocks
  4. Destroy rocks before they hit the ground!
